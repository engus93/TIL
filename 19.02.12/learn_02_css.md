# CSS

### 1. CSS란?

**※ Cascading Style Sheets의 약자입니다.**

- C(Cascading) : 계속되는 / 연속되는
- S(Style) : 모양 / 방식 /스타일
- S(Sheets의) : 덮여있는

→ `반복적인 모양(방식)의 덮어주는 천(커버) 같은 역할`

ex)

```
h1 {
  color: blue;
  background-color: yellow;
  border: 1px solid black;
}
```

### 2. CSS 동작 방식

```
1. 브라우저가 HTML과 CSS를 DOM으로 변환
2. 이제 문서의 내용과 스타일을 결합
3. 브라우저가 DOM의 내용을 표시
```
